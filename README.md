# 🏠 Bengaluru House Price Prediction using Machine Learning

A Machine Learning project that predicts house prices in Bengaluru based on various property features. The project includes complete data preprocessing, feature engineering, model training, and evaluation using Python and Scikit-learn.

---

## 📌 Project Overview

The objective of this project is to build a regression model that accurately predicts the selling price of residential properties in Bengaluru. The dataset is cleaned, transformed, and processed before training a Linear Regression model.

---

## 🚀 Features

- Data Cleaning and Preprocessing
- Handling Missing Values
- Feature Engineering
- Categorical Encoding
- Label Encoding
- Train-Test Split
- Linear Regression Model
- Model Evaluation
- House Price Prediction

---

## 📂 Dataset

The project uses the **Bengaluru House Price Dataset**, which contains information such as:

- Area Type
- Availability
- Location
- Size (BHK)
- Total Square Feet
- Number of Bathrooms
- Balcony
- Price
